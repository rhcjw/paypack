## 项目概况

**PayPack** — AI Agent 通用支付中间件 / Dify 插件
- 核心库 + Dify 插件: **开源**（免费引流）
- 云后端 (Cloud SaaS): **私有**（收费服务）
- GitHub: https://github.com/rhcjw/paypack
- Gitee: https://gitee.com/rhcjw_com/paypack
- PyPI: `pip install langchain-paypack`
- 当前版本: **v0.6.0** ✅

---

## 商业模式

```
核心库开源 ──→ 开发者自己搭（麻烦、费时）
                  │
                  └──→ 选择 PayPack Cloud（付费 API）
                         ├── 支付宝支付
                         ├── 查单 / 退款
                         └── 无需管私钥，开箱即用
```
